Jekyll-Uno - a minimal, responsive theme for Jekyll

Jan 9, 2016
jekyll

I decied to convert my site over to Jekyll from Ghost and host it with Github Pages. This allows all my content to live in static files and be built automatically by GitHub. My hope is that with this simplification (and automation) I might actually post a little more this year.

During the conversion process I decided that I liked the theme I was currently using with Ghost but found it was not available for Jekyll yet. So I decided to port the theme over and share.

Even if this doesn't motivate me to post more at least a new Jekyll theme came out of it. Enjoy!

Jekyll-Uno Theme

More details can be found on the theme's project page.
